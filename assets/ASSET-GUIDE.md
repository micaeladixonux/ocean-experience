# Ocean Experience - Asset Placement Guide

Place your generated assets in the folders below. 
Follow the exact naming conventions for the code to work correctly.

---

## /landing/

```
ambient.mp4          ← Main landing background video (15-30 sec loop)
ambient.webm         ← WebM version (optional, for better browser support)
```
